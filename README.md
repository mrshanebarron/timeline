# Laravel Design Timeline

Timeline component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/timeline
```

## Usage

### Livewire Component

```blade
<livewire:ld-timeline />
```

### Blade Component

```blade
<x-ld-timeline />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-timeline-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-timeline-views
```
